Setup

Zero to agent OS
in under 10 minutes.

01
Boot or install
Flash the ClawOS ISO to a USB — boots on any x86 machine. Or run the install script on Ubuntu 24.04.
02
First-run wizard
Pick your runtime (Claw Core or OpenClaw), scan a WhatsApp QR, and set a dashboard PIN. Done.
03
Models pull automatically
Ollama downloads gemma3:4b (or qwen2.5:7b for OpenClaw) on first boot. Works CPU-only.
04
Start talking
Message via WhatsApp, use the dashboard, or speak. Your AI is live. All data stays on the machine.

FAQ

Common questions

Does it actually run without internet?
Yes. Claw Core is 100% offline. Ollama runs the LLM locally, Whisper/Piper handle voice locally, and WhatsApp works over LAN. The only time it phones home is to pull model weights on first install and (optionally) sync Pro backups.
What's the difference between Claw Core and OpenClaw?
Claw Core is our native Python ReAct agent — fast, low RAM (works on 8 GB), CPU-only. OpenClaw is the open-source version of Claude's full tool ecosystem (430K lines, 13K+ skills) pre-configured for Ollama — needs 16 GB+ and a stronger model like qwen2.5:7b.
Is my data sent anywhere?
Never, unless you explicitly enable Pro cloud backup (E2E encrypted). All LLM inference, voice, memory, and tool execution happens on your machine. No telemetry, no analytics, no tracking.
Can I run it on macOS?
The dashboard and most services run on macOS. The full OS image (ISO) is Linux (Ubuntu 24.04). macOS users can run ClawOS as a stack via the install script — all features work except the bootable ISO.
What models does it use?
Default: gemma3:4b (great on 8 GB, CPU-only). OpenClaw mode needs qwen2.5:7b or better for tool calling. You can swap to any Ollama-compatible model in Settings → Model. Mistral, Llama 3, Phi-3 — all work.
How is this different from just running Ollama + Open WebUI?
ClawOS is an agent OS, not a chat UI. It has: policy-gated tool execution, a 4-layer memory system, a living knowledge graph, 27+ automation workflows, WhatsApp integration, proactive ambient intelligence, a browser agent, and a skills marketplace — all wired together and working on first boot.
