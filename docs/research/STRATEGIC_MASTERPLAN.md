### Version 0.2.0 — "JARVIS Awakens"
**Theme:** It delights
**Timeline:** 6-8 weeks

#### Feature: Visual Onboarding Wizard
**Purpose:** Solve #1 user complaint ("too hard to set up")

**User flow:**
1. Download installer (DMG/EXE/AppImage)
2. Launch → "Welcome to ClawOS" animated splash
3. Hardware scan animation (detecting RAM/GPU...)
4. Auto-download appropriate models
5. Voice test ("Say 'Hello JARVIS'")
6. Personality selection (JARVIS/FRIDAY/Custom)
7. First conversation
8. "You're all set!" + quick actions (Summarize downloads, Daily briefing, etc.)

**Technical:**
- Tauri v2 (Rust + WebView)
- Bundles Ollama installer
- Validates each step before proceeding
- Creates recovery point if setup fails

**Success metric:** Setup completion rate > 80% (currently ~30% for OpenClaw)

---

#### Feature: JARVIS Desktop App
**Purpose:** Native experience, global shortcut, system tray

**Capabilities:**
- Global shortcut: Cmd/Ctrl+Shift+J (instant wake)

- Voice activation: "Hey JARVIS" (configurable)
- Visual states:
  - Idle: Pulsing blue orb
  - Listening: Expanding green rings
  - Thinking: Orange particles

**Technical:**
- Tauri for cross-platform
- Three.js for 3D orb
- WebRTC for voice streaming
- System APIs for global shortcuts

**Success metric:** Daily active users via desktop app

## Phase 4: Technical Architecture

### Desktop App Stack (Recommended)

| Layer | Technology | Rationale |
|-------|------------|-----------|
| **Framework** | Tauri v2 | Rust + WebView = fast, secure, small |
| **UI** | React + Tailwind | Familiar, component ecosystem |
| **3D** | Three.js | Orb visualization, particle effects |
| **State** | Zustand | Simple, no boilerplate |
| **Voice** | WebRTC + WebSocket | Real-time streaming |
| **Build** | GitHub Actions | Cross-platform releases |

### Alternative: Electron vs Tauri

| Factor | Electron | Tauri | Winner |
|--------|----------|-------|--------|
| Bundle size | 150MB+ | 15MB | Tauri |
| Memory usage | High | Low | Tauri |
| Security | Good | Better | Tauri |
| Ecosystem | Massive | Growing | Electron |
| Team experience | Common | Rust learning | Depends |

**Recommendation:** Tauri v2 for new project
