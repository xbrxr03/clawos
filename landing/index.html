Everything people email us about, answered up front.

Do I need to be technical to use this? +

No. The installer is one line — copy, paste, press enter. The dashboard runs at localhost:7070. If you can click a button, you can use ClawOS. For voice, say "Hey Jarvis" — no config needed.

Does ClawOS store, share, or train on my data? +

No. Everything runs on your hardware. No telemetry. No analytics. Every line of code is AGPL-3.0 — audit it yourself on GitHub. Cloud AI models (if you enable them with your own API key) follow those providers' policies; local models (Ollama) never leave your machine.

What hardware do I need? +

Minimum: any PC, Mac, or Linux box with 8 GB RAM (runs Ollama 3B/7B models, Piper voice, full dashboard). Recommended: 16 GB+ for smooth larger-model inference. A spare laptop, old desktop, or mini PC works. No GPU required for basic use.

What's the difference between free and $10 Premium? +

Premium ($10 once): all 29 workflows unlocked, cloud model profiles (Anthropic/OpenAI/Azure wired), ElevenLabs voice wiring (your API key), CrossEncoder reranking, premium skill packs, priority Discord.
